# Aplicación de Predicción de Riesgo Cardíaco

Este repositorio contiene una aplicación web desarrollada con Flask que permite realizar predicciones de riesgo cardíaco utilizando un modelo de Machine Learning entrenado previamente.

## Estructura del proyecto
- app.py → servidor Flask
- templates/index.html → interfaz de usuario
- model/heart_disease_model.pkl → modelo entrenado
- requirements.txt → dependencias
- .gitignore → archivos ignorados por Git

## Cómo ejecutar
1. Crear y activar entorno virtual
2. Instalar dependencias: pip install -r requirements.txt
3. Ejecutar: python app.py
4. Abrir en el navegador: http://127.0.0.1:5000
